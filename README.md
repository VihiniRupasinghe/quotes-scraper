\f0\fs24 \cf0 # Quotes Scraper Project\
\
## Overview\
This project collects quotes, authors, and tags from http://quotes.toscrape.com using Python.\
It demonstrates web scraping, data cleaning, and basic data analysis skills.\
\
## Data Source\
http://quotes.toscrape.com \'97 a public, beginner-friendly website designed for scraping practice.\
\
## Tools Used\
- Python\
- Requests\
- BeautifulSoup\
- Pandas\
\
## Process\
1. Send a GET request to the website.\
2. Parse HTML using BeautifulSoup.\
3. Extract quote text, author names, and tags.\
4. Save data to a CSV file.\
5. Conduct basic analysis: \
   - Count quotes per author\
   - Count frequency of tags\
\
## Results\
- Generated a CSV containing quotes, authors, and tags.\
- Identified authors with the most quotes.\
- Identified most frequent tags for insights into popular themes.\
